Obrerx multimedia, apasionadx por el software libre, exponente sonoro de música electrónica con livecoding, entusiasta e investigadorx de las comunicaciones alternativas y de tecnologías de bajo costo.


Actualmenta trabajo como creativx digital, en el area de comunicaciones de Platohedro, dentro de mis habilidades esta la utilización de wordpres para la contruscción de paginas web dinamicas y gentión de contenidos, html 5 para la maquetación y customización de sitios web.


Realizo procesos creativos para piezas gráficas, publicaciones web entre otros conteidos multimedia por medio de herramientas de código abierto como gimp, inskcape y scribus


Me desenvuelvo con la web 3.0 usando blockchain para diversos objetivos, como servicios de hosting web IFPS, manejo redes descentralizadas mints, creo escenarios 3D para Realidad virtual en Descentraland.


S conceptos básicos sobre manejo de wallets y sé usar herramientas como xmrig para la minería de criptomonedas
